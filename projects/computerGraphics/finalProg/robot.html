README

Arnthor Atli's final project in Computer Graphics fall 2024:

  • Perspective: Switches between perspective and orthographic projection.
  • Look At: Adjusts the camera to focus on the robot model.
  • Change Model: Switches between models, there are 3 in total
  • Animate: Starts a wave animation for the robots, if in ball model then Starts/Stops the ball spinning
  • Zoom: Adjusts the camera zoom level.
  • Robot Z-location: Changes the robot's position along the Z-axis.
  • View Z-location: Adjusts the camera's Z-axis position.

The ground is just 5*7 upscaled picture objects with the Minecraft grass texture on them
